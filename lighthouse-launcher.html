How to run Lighthouse in Chrome DevTools:

  1. Open Chrome DevTools (F12 or Ctrl+Shift+I)
  2. Click the Lighthouse tab (or PageSpeed Insights)
  3. Select categories to test (Performance, Accessibility, Best Practices, SEO)
  4. Choose Desktop device
  5. Click "Analyze page load"
